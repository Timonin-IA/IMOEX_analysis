# Анализ исторических данных индекса Мосбиржи

## Краткое описание
Сбор данных уровней IMOEX за последние ~27 лет через API Мосбирже. Статистический анализ исторических данных. Регрессионная модель.

## Цели 
1. Получить данные по индексу через API Мосбиржи
2. Провести разведочный анализ данных и подготовить их к дальнейшему анализу
3. Провести простейший статистический анализ поведения индекса по дням недели, неделям в году
4. Опционально: найти исторические данные по некоторым макроэкономическим покзателям (инфляция, ключевая ставка, курс доллара)
5. Опционально: разработать линейную регрессионную модель зависимости индекса от макропоказателей
